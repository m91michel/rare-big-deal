## Product Overview

DNSDeck is a powerful DNS management application specifically designed for macOS, simplifying the management of DNS records across multiple cloud providers. Whether you're a developer, system administrator, or managing various domains, DNSDeck provides an intuitive interface to handle all your DNS needs.

NO SUBSCRIPTIONS

Pay once to download; the app is yours, along with a lifetime of free updates. No in-app purchases are added to unlock any core functionality. It is available to access across all devices bound to your Apple Account.

KEY FEATURES

- Multi-Provider Support
Seamlessly manage DNS records across Cloudflare, Vercel, Google Cloud DNS and AWS Route 53 from a single application. Switch between providers effortlessly and maintain consistency across your DNS infrastructure.

- Secure Credential Storage
With iCloud sync support, your API tokens and access keys are securely stored in the iOS Keychain. You need not worry about credential security or losing access across your devices.

- Comprehensive Record Management
Create, edit, and delete all major DNS record types, including A, AAAA, CNAME, MX, TXT, SRV, PTR, CAA, and NS records. Advanced record types are fully supported with intuitive form interfaces.

- Real-Time Updates
Changes are reflected immediately in your DNS provider. There is no waiting, no delays—see your updates take effect instantly across your infrastructure.

- Search and Filter
Quickly find specific domains and DNS records with powerful search functionality. Navigate large DNS zones effortlessly with intelligent filtering.

- Native iOS Experience
Built specifically for iOS with a native SwiftUI interface. It supports dark mode and keyboard shortcuts and follows Apple's Human Interface Guidelines for a familiar experience.

- Professional Tools
TTL management, proxy status control for Cloudflare, priority settings for MX records, and comprehensive validation ensure your DNS configuration is always correct.

SUPPORTED PROVIDERS

- Cloudflare - Full API integration with proxy status management
- Amazon Route 53 - Complete hosted zone and record management
- Vercel - Complete deployment and DNS management
- Google Cloud DNS - Comprehensive DNS zone and record management

PERFECT FOR

- Web developers managing multiple domains
- System administrators handling DNS infrastructure  
- DevOps teams requiring reliable DNS management
- Anyone who needs professional DNS record control

SECURITY & PRIVACY

DNSDeck takes security seriously. All credentials are stored in iOS Keychain, network communications use HTTPS with certificate validation, and no sensitive data is logged or stored in plain text.

You can start with your existing Cloudflare API tokens or AWS credentials in minutes. DNSDeck makes DNS management simple, secure, and efficient.
